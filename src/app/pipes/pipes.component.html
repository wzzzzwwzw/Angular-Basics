Ejemplos de pipes

La cantidad de dinero es: {{ mymoney | currency:'€':'symbol':'1.0-2'}}
El número original es: {{ mynumber }}
El número con un decimal es: {{ mynumber | number:'1.0-1' }}
La fecha es: {{ mymsseconds | date:'dd/MM/yyyy' }}
El objeto json es: {{ myobject | json }}
La cadena convertida a minúsculas: {{mystring | lowercase}}
La cadena convertida a mayúsculas: {{mystring | uppercase}}
La cadena convertida con "titlecase": {{mystring | titlecase}}
